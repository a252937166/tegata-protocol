# Security policy

This is a hackathon prototype on HashKey Chain testnet (the mainnet deployment
anchors proof state only; no user funds are ever held by the protocol — HSP
wallet-settling means the payer's own wallet settles every transfer).

- Threat model: [docs/THREAT_MODEL.md](docs/THREAT_MODEL.md)
- Known limitations: [docs/KNOWN_LIMITATIONS.md](docs/KNOWN_LIMITATIONS.md)

To report a vulnerability, open a GitHub issue with the `security` label (no
sensitive PoC details in public — request a private channel and we will
provide one). Demo keys and sandbox credentials are rotated after the
hackathon; nothing in this repository grants access to real assets.
